# KING-NASSIR_VS_OBAMA
Un enfrentamiento entre king nassir y obama 
- para este codigo cree 4 clases, estas dichas clases son el ataque, la defensa y king nassir y obama, cada clase tiene sus propios atributos y metodos, el ataque tiene un metodo para atacar y el defensa tiene un metodo para defenderse, king nassir y obama tienen atributos como la vida y la bateria, ademas de un metodo para verificar si estan activos o no. el codigo simula un enfrentamiento entre ambos personajes y muestra el resultado final.
